東方封魔再録 ~ the Return of Eastern Wonderland

(Touhou Fuumasairoku ~ the Return of Eastern Wonderland, lit. "Eastern Repeat of Demon-Sealing Record")

Genre: Shoot-'em-up (Bullet Hell)
Engine: Touhou Danmakufu ph3sx
Project lead: WishMakers

The Return of Eastern Wonderland is a fan-made "reimagining" of the second Touhou game, Story of Eastern Wonderland. This game was the project that brought together Analogue Reverie as a circle.

Currently, the game is still in heavy development. Please wait warmly for updates.
